# 🏗️ IronVision Forecater 
---
#  Problem Statement

Through discussions with the Ironsite team, we learned that **nearly 20%** of a construction workday is lost to material transport caused by reactive and inefficient planning. 
While spatial intelligence systems already capture rich site footage, that data is largely used only for documentation. 
We built a system that converts construction video into forward-looking predictions of upcoming tasks and material needs, enabling proactive staging, reducing idle labor, and improving overall site efficiency

# Problem Solution

IronVision Forecaster is a multi-stage AI system that transforms raw construction site video footage into structured progress intelligence, next-day task plans, material forecasts, and executive-ready reports. 
By combining vision models with structured blueprint logic, it analyzes what work has been completed and predicts what should happen next. 
The system converts passive site footage into actionable operational insights that improve planning, reduce material transport inefficiencies, and support data-driven decision-making.

# Tech Stack

* VLM Used: Groq — llama-4-scout-17b-16e-instruct (Llama 4 Scout with vision capability, used for analyzing construction video frames)
* LLM Used: Groq — llama-3.3-70b-versatile (Llama 3.3 70B, used for reasoning about the blueprint + summary and generating the next-day plan)
* Frontend: Streamlit
* Backend: Python
* GPU: NVIDIA RTX 3060
